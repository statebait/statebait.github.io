# Personal Website

Built with [Svelte](https://svelte.dev/) and [Tailwind CSS](https://tailwindcss.com/).

### Lighthouse CI

- Install Lighthouse CI's cli globally:

```
yarn add global @lhci/cli
```

- Then run it like this:

```
yarn build && lhci autorun
```
